About Jellytool

What is Jellytool?

Jellytool is a project I came up with while learning python. I have my own jellyfin server and have accounts on a few other peoples servers and I figured it could be useful to automate some things. Currently jellyfin automates the user deletion and suspension process by synchronizing the database, querying it, and checking for either last login or last watched title and comparing that to a set time or default of 30 days to suspend or delete a user based on input. it is a command line tool and comes with a bash script to automate it via cron. Many more features are planned and I am taking requests from the community. Check the subreddit at /r/jellytool for requests and discussion.
